Mediator + module паттерн

Результат выполнения: в консоли

Задача: Создать обьект который будет распределять связи/задачи с другими обьектами (которые не должны быть связаны между собой)

Практический пример:

Есть "Прораб"(медиатор), укладчик кирпича, установщик окон, установщик крыши. Также есть склад кирпича и склад для крыши.

Нужно:

  • Началась постройка из кирпича
  • Если каркас дома готов к установке окон, то начать установку окон и продолжить дальше постройку дома
  • После окончания постройки "короби" начать установку крыши

Входные данные: Склад из 5*100 кирпичей(brickStorage), roofStorage: 3 детали . Установка окон начинается после изпользования 200 кирпичей.
